# Florida 2-15 Exam Study Tools

Interactive HTML study tools for the Florida 2-15 (Health & Life including Annuities & Variable Contracts) insurance licensing exam.

Each file is a self-contained single-page app with **flashcards** and a **practice test**, built from the Xcel pre-licensing course review notes.

## Files

| File | Chapter | Topics |
|------|---------|--------|
| `ch1_study_tool.html` | Chapter 1 | Basic Principles of Life and Health Insurance |
| `ch2_study_tool.html` | Chapter 2 | Risk, Hazards, Perils & Methods of Handling Risk |
| `ch3_study_tool.html` | Chapter 3 | Insurance Contracts — Elements, Features & Legal Principles |
| `ch4_study_tool.html` | Chapter 4 | Life Insurance Products |
| `ch5_study_tool.html` | Chapter 5 | Life Insurance Policy Provisions, Riders & Exclusions |

## How to Use

1. Download or clone this repo
2. Open any `.html` file directly in your browser — no server needed
3. Use the **Flashcards** tab to drill key terms and definitions
4. Use the **Practice Test** tab to simulate exam questions with instant feedback and explanations

## Exam Info

- **Exam:** Florida 2-15 Health & Life (including Annuities & Variable Contracts)
- **Questions:** 165 total (150 scored), 70% passing threshold
- **Time:** 2 hours 45 minutes
- **Administered by:** Pearson VUE
